# Link-Like UI

A certain UI library that looks like Link-Like-OS.

## Usage

```tsx
import { LoadingOverlay } from "react-components-starter/Loading";
import { TabRoot } from "react-components-starter/Tab";
import { Button } from "react-components-starter/Button";
```

## Development

- Install dependencies:

```bash
pnpm install
```

- Run the playground:

```bash
pnpm run play
```

- Run the unit tests:

```bash
pnpm run test
```

- Build the library:

```bash
pnpm run build
```
